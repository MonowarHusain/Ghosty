# 👻 Ghosty - Anonymous Messaging Platform

Ghosty is a sleek, modern, and lightweight "NGL-style" anonymous messaging application. It allows users to receive messages, feedback, or "ghostly" secrets directly in their **Discord** channel via Webhooks.

## ✨ Features

- **🛡️ 100% Anonymous:** No login, tracking, or registration for senders.
- **⚡ Instant Delivery:** Messages pop up in your Discord server as soon as they are sent.
- **📱 Responsive UI:** Mobile-first design that looks great on any screen size.
- **☁️ Serverless Ready:** Pre-configured for deployment on Vercel or Render.
- **🔒 Secure:** Private webhook URLs are managed via environment variables (no leaks!).

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, CSS3 (Modern Flexbox/Glassmorphism), Vanilla JavaScript.
- **Backend:** Node.js, Express.js.
- **API:** Discord Webhook API.
- **Deployment:** Vercel.
